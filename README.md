** Student Grade Calculator**

A beginner-level Python program that calculates a student's academic results from marks entered by the user.

**Features**

* Accepts marks for multiple subjects
* Validates marks between 0 and 100
* Calculates the average mark
* Identifies the highest and lowest marks
* Assigns a letter grade
* Displays an overall pass or fail result
* Handles invalid user input

## Technologies

* Python 3

## Programming Concepts Demonstrated

* Functions
* Dictionaries and lists
* Loops
* Conditional statements
* User input
* Exception handling
* Basic calculations

## How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run:)
python grade_calculator.py

## Grading Rules

| Average  | Grade |
| -------- | ----- |
| 90–100   | A     |
| 80–89    | B     |
| 70–79    | C     |
| 60–69    | D     |
| Below 60 | F     |

A student passes only when every subject has a mark of at least 40.
